__construct()

__construct(  $args)

The Object's constructor A. One argument string/int can be the primary key of the table. This will also load the row into the object.

e.g new TableObjectChild(4); B. As many arguments as there are properties of the object, in the correct order. Any auto_incrementing column must be set to false e.g new TableObjectChild(false,'Joe','Soap','joe.soap@gmail.com'); C. An array of as many arguments as there are properties of the object. Any auto_incrementing column must be set to false. Must be in the correct order. e.g (php5.5) new TableObjectChild([false,'Joe','Soap','joe.soap@gmail.com']); e.g (php5.4) new TableObjectChild(new array(false,'Joe','Soap','joe.soap@gmail.com'));
